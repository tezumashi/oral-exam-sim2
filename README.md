# 二等無人航空機操縦士 修了審査 口述審査シミュレーション

二等（および一等）無人航空機操縦士の修了審査における口述試験を、ひとりで声に出して練習するためのWebアプリです。ビルド不要の静的サイト（HTML/CSS/JSのみ）なので、GitHub Pagesでそのまま公開できます。

## 使い方

1. 練習したい範囲（飛行前点検／口述飛行後点検／事故・重大インシデント）にチェックを入れて「練習を開始する」を押します。
2. 試験員役の質問が表示されたら、口に出して回答します。
3. 回答できたら **Space** キー（または「正解を表示する」ボタン）で正解例を表示します。
4. 自己採点し、**Enter** キーで「正解」、**Backspace** キーで「不正解」を記録します。
5. **Space** キーで次の問題に進みます。
6. 全問終了すると結果画面が表示され、間違えた問題だけをまとめて復習できます。
7. 学習履歴はブラウザのlocalStorageに保存され、「学習履歴を見る」からいつでも振り返れます（端末・ブラウザ内のみに保存され、外部には送信されません）。

## ローカルでの確認方法

Node.js等は不要です。任意の静的サーバーで配信するだけで動作します。

```bash
python3 -m http.server 8000
```

その後 `http://localhost:8000` を開いてください。

## GitHub Pagesで公開する方法

1. GitHubで新しいリポジトリを作成します（Public推奨。README等は追加せず空のリポジトリでOK）。
2. このフォルダの内容をそのリポジトリにpushします。
3. リポジトリの **Settings → Pages** を開き、「Build and deployment」の Source を **Deploy from a branch** に、Branch を **main** / フォルダを **/(root)** に設定して保存します。
4. 数分後、`https://<ユーザー名>.github.io/<リポジトリ名>/` でアプリが公開されます。

## ファイル構成

- `index.html` — 画面構造
- `style.css` — スタイル
- `app.js` — 出題・採点・履歴のロジック
- `data.js` — 問題・正解例データ（このファイルを編集すれば問題を追加・変更できます）
